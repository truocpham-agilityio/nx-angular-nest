# NxAngularNest

This project was generated using [Nx](https://nx.dev).

## Tech Stacks

- TypeScript
- Nx is a set of Extensible Dev Tools for Monorepos
- Angular framework
- NestJS framework

## Installation

```bash
# Install dependencies
$ npm i

# Or
$ yarn
```

## Run on locally

Run api

```bash
# Development api
$ npm run start:api
```

Run app

```bash
# Development angular app
$ npm run start:app
```
